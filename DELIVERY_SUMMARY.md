## 🚀 Quick Start (5 Minutes)

```bash
# 1. Setup Database
mysql -u root -p < database_setup.sql

# 2. Install Dependencies
cd "g:\web technology\html"
npm install

# 3. Configure Environment
cp .env.example .env
# Edit .env with your MySQL password

# 4. Start Server
npm run dev

# 5. Open Browser
# http://localhost:3000
```
